NumPy and Exercises 📊🐍

This project contains Jupyter Notebooks demonstrating NumPy array operations, indexing techniques, and practical exercises.
It is designed for beginners who are learning NumPy fundamentals and want hands-on examples.

📂 Project Files
1️⃣ Array_Indexing.ipynb

This notebook demonstrates NumPy array indexing and slicing techniques.

Topics Covered:
Creating NumPy arrays using np.arange()
Accessing array elements using indexing
Array slicing techniques
Modifying sliced arrays
Understanding views vs copies in NumPy
Example Concepts:
Single element access
Range slicing
Array sub-selection
----------------------------------------------------
2️⃣ Array_Operations.ipynb

This notebook covers arithmetic operations on NumPy arrays.

Topics Covered:
Creating NumPy arrays
Performing arithmetic operations:
Addition (a1 + a2)
Subtraction (a1 - a2)
Multiplication
Division
Element-wise operations
Using NumPy for mathematical computations
Example Concepts:
Vectorized operations
Array-to-array calculations
Efficient numerical computation
----------------------------------------------------
3️⃣ Exersises.ipynb

This notebook contains practice exercises using NumPy, including a Sudoku validation example.

Topics Covered:
Creating 2D NumPy arrays
Row and column operations
Using np.sum() with axis
Checking Sudoku validity using row and column sums
Applying NumPy in real-world logical problems
